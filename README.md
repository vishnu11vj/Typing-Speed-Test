# ⌨️ Typing Speed Test

A clean typing speed test with real-time WPM and accuracy tracking.

## Features
- 60-second countdown timer
- Real-time WPM (words per minute) calculation
- Accuracy percentage tracking
- Color-coded correct and wrong characters
- Results screen with WPM, accuracy, and error count
- Restart anytime

## Built With
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
Clone the repo and open `index.html` in your browser.
